# EchoWeave Studio - Core PRD

## Architecture
- Framework: Next.js (App Router)
- Database: PostgreSQL (via Supabase)
- Styling: TailwindCSS

## Modules
- **Core Platform**: Authentication, Project management, Dashboard, Database schema (users, projects, chapters).
- **Collaboration Engine**: Chapter Builder (rich text editor), Conversations (chat/transcript area for ideas).

## Feature Sets
- Secure user login for co-authors (Xavier and Natalie).
- Real-time collaboration on text documents.
- Activity feed for tracking updates.
- Commenting and revision history.

## Implementation Tasks
- Scaffold Next.js application.
- Set up Supabase auth and database schema.
- Build Dashboard UI.
- Implement rich text Chapter editor.
- Implement chat-style conversations module.
